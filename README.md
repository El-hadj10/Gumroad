# Gumroad - Fichiers Sensibles et Internes

Dépôt privé pour stocker les fichiers sensibles, personnels et internes des produits Gumroad.

## Structure

### nav-starter/

Fichiers exclus du package public:

**Fichiers Personnels**
- `GITHUB_PROFILE_README.md` - Profil GitHub personnel
- `.vscode/launch.json` - Configuration IDE personnelle
- `.vscode/mcp.json` - Configuration IDE personnelle

**Étapes/Roadmap**
- `LAUNCH_ROADMAP.md` - Roadmap de lancement (anglais)
- `VENDRE_ETAPES.md` - Étapes de vente (français)

**Documentation Interne**
- `PRE_LAUNCH_CHECKLIST.md` - Checklist avant publication (usage personnel)
- `SALES_PITCH.md` - Stratégie de vente et pricing
