# Gumroad - Fichiers Sensibles

Dépôt privé pour stocker les fichiers sensibles et personnels des produits Gumroad.

## Structure

- `nav-starter/` - Fichiers sensibles du projet nav-starter
  - `GITHUB_PROFILE_README.md` - Profil GitHub personnel
  - `LAUNCH_ROADMAP.md` - Planning de lancement personnel
  - `VENDRE_ETAPES.md` - Étapes de vente personnelles
  - `.vscode/` - Configurations IDE personnelles

**Note:** Ce dépôt n'est pas inclus dans les packages vendus sur Gumroad.
